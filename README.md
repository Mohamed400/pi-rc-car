# Raspberry Pi RC Car

A Raspberry Pi 4 based RC car project controlled using a PS4 controller over Bluetooth.

## Current Status

- Raspberry Pi 4 setup complete
- Raspberry Pi OS installed
- SSH configured
- Git repository configured
- PS4 controller paired over Bluetooth
- Pygame controller interface working
- Controller axis mapping in progress

## Planned Features

- Steering control
- Motor control
- Live camera streaming
- Teleoperation from laptop
- Autonomous driving experiments

## Hardware

- Raspberry Pi 4
- WLtoys 144011 RC Car
- Raspberry Pi Camera
- DC-DC Buck Converter
- PS4 Controller
